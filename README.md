# Saudi Architecture Website
A simple HTML and CSS web project that highlights the beauty, diversity, and identity of Saudi Arabian architecture.
It provides structured information about traditional and modern architectural styles across different regions of the Kingdom.

## Features
- Three linked pages:
  - **Home Page** – Overview of Saudi architecture
  - **Map Page** – Displays architectural distribution across regions
  - **Design Guidelines** – Explains styles and visual principles with an illustrated table
- Responsive navigation menu
- Clean and consistent color palette inspired by Saudi heritage
- Simple, static design built using only HTML and CSS

## Pages Overview
1. `SaudiArchitecture.html`
  Introduces the concept of Saudi architecture and includes a quote from HRH Prince Mohammed bin Salman emphasizing the harmony between heritage and modernity.
2. `DesignGuidelines.html`
  Describes design principles and lists 19 distinct architectural identities across Saudi Arabia with regions and sample icons.
3. `map.html`
  Presents an interactive visual map (linked through the navigation bar) to explore architecture geographically.

## Technologies Used
- HTML5
- CSS3
